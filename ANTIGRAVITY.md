# Global Typography Rules

Whenever you generate or AI generate code for this project, you **MUST** strictly adhere to the following typography guidelines:

## Headings
- **Font Family**: 'Roboto Slab', serif
- **Usage**: All headings (`h1` - `h6`), emphasis text, and important UI elements that need visual weight.
- **Weights**: Regular (400), Semi-bold (600), Bold (700)

## Body Text
- **Font Family**: 'neuzeit-grotesk', Helvetica, Arial, sans-serif
- **Usage**: Body text, UI components, buttons, labels, paragraphs, and all interactive elements.
- **Weights**: Regular (400), Semi-bold (600), Bold (700)
