# Plan: Mobile Responsive - Rank Predictor

**Requirement ID:** REQ-6-MOBILE
**Target Path:** `/rank-predictor`

## Context
The Rank Predictor is a form-heavy tool where users input scores to see projected ranks based on past data. The input forms and the resultant data visualization must accommodate narrow screens.

<tasks>
  <task>
    <description>Modify the main layout container in `src/RankPredictor.jsx` so that the form wrapper spans 100% of the mobile screen (`w-full px-4`).</description>
    <expected_outcome>Form inputs for marks/percentile do not look squeezed.</expected_outcome>
  </task>
  <task>
    <description>Ensure any resulting cards or charts (predicting the college cutoff or rank range) display properly underneath the form rather than adjacent to it.</description>
    <expected_outcome>The predicted output is immediately visible below the inputs without zooming or horizontal scrolling.</expected_outcome>
  </task>
  <task>
    <description>Set inputs to trigger the correct native numeric keyboard on mobile devices (`type="number"` or `inputMode="numeric"`).</description>
    <expected_outcome>Simplified numeric data entry for mobile users.</expected_outcome>
  </task>
</tasks>
